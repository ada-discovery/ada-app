@import org.incal.play.controllers.WebContext @import org.incal.play.controllers.WebContext._ @import views.html.documentation.core @import views.html.documentation.section @()(implicit context: WebContext) @core("Data Set Import") { @section(None, None, 0, false) {

To create, edit, delete, schedule, or execute a data set import, go to Admin->Data Set Imports. Ada provides three file-based data import adapters:

  • CSV
  • JSON
  • tranSMART

and three RESTful-based:

  • REDCap
  • Synapse
  • eGAIT

} {
Note
Data sets can be imported only by admins! If you have data that you believe should be imported to (a specific instance of) Ada, contact your admin.
} }
