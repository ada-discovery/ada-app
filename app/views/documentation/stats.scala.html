Ada offers several analytical/statistical methods, which can be easily applied to any data set. Each method has its own screen producing various visualizations/widgets for the selected fields.

Ada currently provides the following stats:

  • Categorical and numerical distributions*
  • Box plots*
  • Cumulative counts*
  • Scatters*
  • Correlations: Pearson and Matthews
  • Heatmaps: 2D aggregations (e.g., count, mean, and max)
  • Descriptive/basic stats (e.g., mean, variance, std, and count)*
  • Independence tests (Chi-square and ANOVA)
* - supports "group by"
